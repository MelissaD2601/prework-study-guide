Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element is a container which stores metadata.
  • Metadata provides information about data.
  • HTML is Hypertext Markup Language.
  • Content visible to users can be in the form of text, images.

CSS

  • CSS provides the style of the webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to the main branch
  • Git is version control software.
  • Git allows users to make changes to the code being edited at various points during editing, which allow the user to revert to previously saved versions of code if necessary.
  • It is recommended to save changes often which will help one understand which version of code one may want or need to revert to.

JavaScript

  • JavaScript helps users interact with the webpage.
