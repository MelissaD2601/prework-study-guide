HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element is a container which stores metadata.
  • Metadata provides information about data.
  • HTML is Hypertext Markup Language.

CSS

  • CSS provides the style of the webpage.
  • Cascading Style Sheets (CSS) is a programming language which defines how webpage content is presented to users.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can refer to a color by its hexadecimal code, or hex #.
  • A border assigns the width and color of a border.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to the main branch
  • Git is version control software.
  • Git allows users to make changes to the code being edited at various points during editing, which allow the user to revert to previously saved versions of code if necessary.
  • It is recommended to save changes often which will help one understand which version of code one may want, or need to revert to.

JavaScript

  • JavaScript helps users interact with the webpage.
